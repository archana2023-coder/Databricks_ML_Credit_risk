# Databricks_ML_Credit_risk
This project demonstrates an end-to-end machine learning pipeline using Databricks, PySpark, Delta Lake, and Feature Store concepts.
## Problem Statement
Predict the probability of credit default using customer and transaction features.

## Tech Stack
- Databricks
- PySpark
- Delta Lake
- Databricks Feature Store
- MLflow

## Pipeline
1. Data Ingestion
2. Feature Engineering
3. Feature Store
4. Model Training
5. Evaluation
